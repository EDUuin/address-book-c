Project Description:

    The Address Book is a program to store and manage contact details such as name, phone number, and email. 
    It allows users to add, search, edit, delete, and list contacts, with all data saved in a CSV file for future use. 
    The program includes input validation, alphabetical sorting, and a simple menu-driven interface for easy navigation.

Key Features:

    # Add new contacts with input validation.
    # Search contacts by Name, Phone Number, or Email ID.
    # Edit or delete existing contacts.
    # List all contacts in alphabetical order.
    # Save and load contact data automatically using a CSV file.
    # Handles multiple matches during search operations.
    # Menu-driven interface with clear, formatted output.

File Handling Functions

---> saveContactsToFile()
---> loadContactsFromFile()
     Manage saving and loading of contacts in a CSV file, ensuring data is not lost between sessions.

Display and Sorting Functions

---> listContacts()
     Used to display all contacts neatly and sort them alphabetically for easy viewing.

Contact Functions

---> createContact()
     ask number of contact to create
     read name, phone number and email address from user and validate at time of entering each parameter
     save in structure Contact with respective createContact
     increment contactCount value

---> searchcontact()
     seacrh contact by name ,phone number and email address and display contact details if found
     if multiple contact found with same name identify with number or email 

---> editcontact()
     edit by name, phone number and email address
     if contact found we can edit the contact 

---> deletecontact()
     delete by name, phone number and email address
     if contact found in list , delete the contact and swap contact from next to previous index
     decrement contact count by one

Validation Functions

---> valname()
     validate the entered name contain upper/lower case alphabet and space
---> valnum()
     validate the entered number have 10 digits
---> valmail()
     validate the entered mail atleast have a lowercase alphabet and "@gmail.com" at end
     and only allows lowercase alphabet, digits and '_' before "@gmail.com"

Enter contact functions

---> enter_name()
     fun. to enter name from user
---> enter_number()
     fun. to enter phone number from user and check number is existing in contact list
---> enter_email()
     fun. to enter email address from user and check email is existing in contact list

Search / Edit / Delete functions

---> search_name()
     search name and list contact(multiple) if found in the contact list 
     if multiple contacts are there identify contact with number or email option will apppear
     and if edit or delete feature is enabled it will support that option 
---> search_number()
     search number and list contact if found in the contact list
     and if edit or delete feature is enabled it will support that option 
---> search_mail()
     search email and list contact if found in the contact list
     and if edit or delete feature is enabled it will support that option
